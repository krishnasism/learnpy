# learnpy

Building a rustlings inspired python hands on coding journey

## Local
```zsh
# Create a venv
uv venv

# Activate venv
source .venv/bin/activate

# Install deps
uv pip install -e .

# Install dev deps
uv pip install -r requirements-dev.txt

# Run
learn-py start
```


## Tests
```zsh
pytest tests/
```
